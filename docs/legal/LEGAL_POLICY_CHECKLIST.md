# LEGAL POLICY CHECKLIST

기준일: 2026-04-21
기준 회사명: 빵빵카 주식회사

## 이미 잠근 기준
- 빵빵카 주식회사가 결제까지 수행하는 직접 계약 당사자
- 예약 흐름은 예약 요청 → 결제 진행 → 결제 완료 시 예약 확정
- 회사는 고지한 사유에 따라 예약 요청 단계 또는 결제 완료 이후에도 예약을 거절하거나 취소할 수 있음
- 운전자 연령 기준은 차량/계약서 기준 만 21세 또는 만 26세
- 면허 취득 1년 이상
- 외국인은 한국 운전면허증 소지자만 대여 가능
- 국제운전면허증만으로는 대여 불가
- 제2운전자는 동일 조건 충족 시 1인 추가 가능
- 현장 면허 확인 실패 또는 예약자 본인 확인 실패 시 배차 불가 가능
- 위 경우 취소요금 부과 또는 환불 제한/불가 가능

  - 승합: 1,000만원 / 50만원
  - 수입, 슈퍼카: 2,000만원 / 100만원
  - 캠핑카: 500만원 / 50만원

  - 단독사고 보장
  - 단, 사고 발생 즉시 회사로 연락하여 사고 경위가 확인되어야 함

- 면책 제한 사유: 1차 확정

  - 무면허운전
  - 약물운전
  - 등록되지 않은 운전자 운행
  - 사고 미신고 또는 지연신고
  - 고의 또는 중대한 과실
  - 사고 후 현장 이탈
  - 기타 법령 및 계약 위반으로 보험 또는 면책 적용이 제한되는 경우
- 추가 검토 후보: 전부 확정
  - 차량 도난 시 키 관리 소홀 또는 문 미잠금 등 이용자 과실이 있는 경우

- 사고 후 차량 회수 / 대차 / 휴차보상료 세부 기준: 확정

  - 대차가 제공되는 경우 대차비용이 발생함

- 사고 발생 시 이용자 의무: 확정
  - 사고 발생 즉시 회사에 연락해야 함
  - 사고 경위를 설명해야 함
  - 사진 등 현장자료 제출에 협조해야 함
  - 필요 시 경찰 신고에 협조해야 함
  - 임의수리 및 임의합의는 금지됨
  - 회사가 지정한 곳 외에서 진행한 수리는 수리로 인정되지 않을 수 있음
  - 위 경우 수리비용이 발생할 수 있음

### 2. 취소 / 환불
상태: 일부 확정
- 무료 취소 구간: 대여 72시간 전까지 전액 환불
- 부분 환불 구간:
  - 대여 72시간 전 ~ 24시간 전: 90% 환불
  - 대여 24시간 전 ~ 5시간 전: 70% 환불
  - 대여 5시간 이내 ~ 대여 시작 전: 50% 환불
- 환불 불가 구간: 대여 시작 이후 / 노쇼
- 회사 사유 취소 시 환불 기준: 확정
  - 회사는 대체차량을 우선 안내할 수 있음
  - 취소 시 전액 환불함
  - 추가 보상은 없음
- 본인 확인 실패 / 면허 확인 실패 / 노쇼 시 환불 기준: 확정
  - 본인 확인 실패 시 환불 불가
  - 면허 확인 실패 시 환불 불가
  - 노쇼 시 환불 불가
- 조기반납 환불 기준: 확정
  - 조기반납 시 환불 없음

### 3. 이용 제한 / 예약 취소 사유
상태: 확정
- 회사는 아래 사유가 있는 경우 예약 요청을 거절하거나, 예약 확정 이후에도 예약을 취소할 수 있음
  - 면허 또는 연령 조건 미충족
  - 예약자 본인 확인 실패
  - 결제 이상 또는 결제 승인 실패
  - 허위정보 입력
  - 차량 부족, 차량 고장, 사고, 정비 필요 등으로 정상 배차가 어려운 경우
  - 연락두절
  - 과거 이용 이력, 사고 이력, 약관 위반 이력 또는 회사의 운영상·안전상 판단에 따라 제한이 필요한 경우
  - 천재지변, 재난, 교통통제, 행정명령 등 불가항력 사유가 있는 경우

### 4. 딜리버리 / 배차 / 반차
상태: 일부 확정
- 가능 지역 문구: 미기재
  - 딜리버리 가능 지역은 위치 선택 시 노출되는 범위를 기준으로 운영
  - 약관/정책 문서에는 별도 지역 문구를 두지 않음
- 시간 기준: 안내 기준
  - 운영 시간 및 가능 여부는 화면 또는 안내 채널 기준으로 고지
  - 약관/정책 문서에는 고정 시간 기준을 두지 않음
- 비용 기준: 안내 기준
  - 딜리버리 비용은 화면 또는 안내 채널에서 고지되는 기준에 따름
  - 약관/정책 문서에는 고정 요금 기준을 두지 않음
- 주소 오기입 / 연락두절 / 진입불가 시 처리: 1차 확정
  - 고객 책임 사유로 정상 배차가 어려운 경우 추가 비용이 발생하거나 서비스가 제한될 수 있음
  - 상황에 따라 배차가 취소될 수 있으며, 이미 출발한 이후에는 환불이 제한될 수 있음
  - 딜리버리 배차지는 차량 인도 및 주차가 가능한 장소여야 함
  - 역, 도로변, 단기정차 구역 등 즉시 이동이 필요하거나 안전한 주차 및 차량 인도가 어려운 장소에서는 배차가 제한될 수 있음
- 반차 변경 / 지연 기준: 확정
  - 고객 요청에 따른 반차 장소 또는 시간 변경은 가능 여부 확인 후 처리함
  - 변경 시 추가 비용이 발생할 수 있음
  - 고객 지연으로 대기 시간이 길어지는 경우 추가 비용이 발생하거나 배차가 취소될 수 있음
  - 회사 또는 기사 일정상 변경 요청이 어려운 경우 요청이 거절될 수 있음
  - 반납 딜리버리도 서비스 특성상 일정 지연이 발생할 수 있음
  - 반납 장소 역시 차량 주차 및 인수가 가능한 장소여야 함
  - 안전한 주차 및 차량 인수가 어려운 장소에서는 반납 서비스가 제한될 수 있음

  - 다만, 관계 법령에서 별도 보관기간을 정한 경우 해당 법령 기준에 따름
- 동의 구조: 확정

  - 개인정보 수집·이용 동의
